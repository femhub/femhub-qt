FEMhub QT
=========

Install PySide
--------------

For Ubuntu::

    sudo add-apt-repository ppa:pyside
    sudo apt-get update
    sudo apt-get install python-pyside

For other distributions, see here:
http://developer.qt.nokia.com/wiki/PySideBinariesLinux

Run FEMhub QT
-------------

In Ubuntu, in the current directory::

    ./femhub-qt
